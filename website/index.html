Unmixing entangled lines of text with a GAN

- Goal: separate 2 lines in contact in a text

- Method: a GAN that takes as input a 3 channel image of the lines to separate and outputs a 3 channel image with, for every channel:

1) the first line

2) the second line

3) the background


Context

The idea behind this project is that lines in contact can be difficult to correctly decipher. The generation of the two lines, separated, is a possible solution to this issue. For example, it could be used before an OCR algorithm to make text recognition easier. This project may therefore be a step toward improving automatic document analysis.

GAN architecture

I use a GAN architecture inspired by Pix2Pix with a U-Net generator and a PatchGAN.
